# hubot-insult-bot

insults

See [`src/insult-bot.coffee`](src/insult-bot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-insult-bot --save`

Then add **hubot-insult-bot** to your `external-scripts.json`:

```json
[
  "hubot-insult-bot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
